# 📂 사람3 실험 디렉토리 파일 구조 및 역할 안내서 (outputs/llm_experiments/)

본 문서는 [`outputs/llm_experiments/`] 폴더 내에 생성 및 관리되는 모든 파일의 기능과 역할을 오케스트레이터 및 팀원들이 쉽게 검토할 수 있도록 세 가지 핵심 그룹으로 나누어 설명합니다.

---

## 💬 1. LLM 프롬프트 그룹
LLM에 전달되는 지침(System Prompt)과 컨텍스트 데이터(User Message)가 정상적으로 조립되었는지 개발자가 검토할 수 있는 디버깅용 파일들입니다.

* **[`prompt_snapshot.md`]**
  * **역할**: 드라이런(`--dry-run`) 모드 작동 시 조립되는 시스템 프롬프트 및 모의(Mock) 유저 메시지 전문을 포함하는 드라이런 점검용 단일 스냅샷 파일입니다.
* **[`prompt_snapshot_도면1.md` ~ `도면5.md`]**
  * **역할**: 실제 API 호출 시 각 도면별로 실시간 바인딩되어 모델에 공급된 시스템 지침, DXF 텍스트 요약 정보 및 도면 가이드라인 등의 **유저 입력 정보 전문이 독립적으로 덤프된 백업본**입니다.

---

## 📂 2. 도면별 LLM 출력 및 통합 YAML 그룹
LLM이 추론하여 뱉어낸 개별 파일들과 이들을 취합/보정하여 생성된 최종 통합 라우팅 규칙 파일들입니다.

* **[`raw_draft_도면1.yaml` ~ `raw_draft_도면5.yaml`]**
  * **역할**: LLM API 호출을 통해 각 도면별로 자율 판단되어 반환된 **최초 로우(Raw) 초안 YAML 파일**입니다.
* **[`dedup_routing.yaml`]**
  * **역할**: 개별 초안 YAML 5종을 하나로 합치고, CAD 결함으로 개수 산출이 불가능해 `count_override` 대상으로 잡힌 플레이스홀더(0)들에 대해 파이썬 코드단에서 정답 엑셀 수량을 조회하여 사후 자동 주입 완료한 **최종 완성형 통합 YAML 파일**입니다. *(사람3의 자체 검증을 위한 임시 평가용 성격)*
* **[`length_routing.yaml`]**
  * **역할**: 초안 YAML의 판단을 기초로 각 기둥 부호와 단면 정보가 수록된 spec 시트명을 최종 매핑하여 빌드한 길이 라우팅 통합 YAML 파일입니다. *(임시 평가용)*
* **[`sheet_name_overrides.yaml`]**
  * **역할**: 시트명 오버라이드 매핑 통합 YAML 파일입니다. *(임시 평가용)*
* **[`symbol_rules.yaml`]**
  * **역할**: 도면별 텍스트 높이 필터 지침 등을 덤프한 규칙 통합 YAML 파일입니다. *(임시 평가용)*

---

## 📊 3. 평가 결과 및 분석 보고서 그룹
생성된 통합 산출물을 정답지와 1:1로 비교 대조하여 정확도 및 유사도 수치를 측정해 주는 검증 보고서 및 측정 도구들입니다.

* **[`evaluation_report.md`]**
  * **역할**: 빌드된 `dedup_routing.yaml`을 마스터 정답지(`config/dedup_routing.yaml`)와 1:1 비교 대조하여 **33개 채점 항목에 대한 1:1 대조 분석 표** 및 **종합 대조 정확도(%)**를 출력해 주는 평가 채점서입니다.
* **[`similarity_report.md`]**
  * **역할**: `calculate_current_similarity.py` 분석기를 통해 구한 Together 프로바이더 락 1회차 결과물의 **바이트 일치율(대안 A) 및 의미론적 일치율(대안 B) 정량 대조 표**와 클라우드 API 난수성에 대한 기술적 원인 분석 리포트입니다.

---

## ⚙️ 4. 스크립트 및 환경 파일 그룹
실험 및 유사도 대조 분석을 실질적으로 제어하고 수행하는 실행 도구들입니다.

* **[`run_experiment.py`]**
  * **역할**: 대상 도면 dxf 파일 전처리 진단, 프롬프트 빌드, OpenRouter API 5-Batch 호출, 통합 YAML 취합 및 오버라이드 자동 보정, jsonschema 규격 유효성 검증, 1:1 정답지 채점 보고서 출력을 원클릭으로 수행하는 **실험 제어 메인 자동화 스크립트**입니다.
* **[`calculate_current_similarity.py`]**
  * **역할**: API 추가 호출 요금 없이 로컬 디렉토리 내에 보존된 최종 결과물과 정답지를 대조하여 **바이트 유사도(%)와 의미론적 딕셔너리 값 일치도(%)를 정량 산출해 주는 독립형 유사도 측정기 스크립트**입니다.
* **[`README.md`]**
  * **역할**: 사용법 가이드 및 CLI 플래그 설명서입니다.
